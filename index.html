YouTube - playlists manager - 1.0

About

I'm a guy who fully uses YouTube's playlists and I'm also passionate about UX (User eXperience). Currently it's relatively complex to perform certain actions with the actual system on youtube.com, such as move a video from a playlist to another or to move several items at the same time. Some other actions requires too many clicks like changing the privacy or the name of the playlist.

So I have decided to make something different, a web application to simplify everything. Now you can edit all your playlists and videos in some few clicks. And even if you don't need to edit anything you can have an eye on all your playlists and their videos. It's really enjoyable to play some music by clicking easily where you want :)

How it's work ?

If you click this button : Login with Google a Google's popup will appear to connect you to your personnal YouTube account directly. After that you will be able to manage your own playlists by rename it and reorder the videos etc. Once you have completed your changes, you will be able to summarize all changes with a diff viewing by clicking this button :  .

When the diff page appear you have the choice to cancel all these changes or save it. On this page you will see different icons etc. you can understand easily with this picture : help-diff.jpg.

Development

This application use the Google's API with the JavaScript gapi client. In other way, the development of this project required several other JavaScript sub-projects including three jQuery plugins :

  • jQuery-selection to enable select multiple items simultaneously with the Ctrl and Shift keys on the keyboard.
  • jQuery-dragndrop to manage the drag and drop of the videos.
  • jQuery-tabs to have some movable tabs.
  • diff.js to make a diff between two arrays, I wrote this to know what has changed.

These three plugins can function independently of each other but once combined it increases their functionality ! For example, it's possible to use jQuery-dragndrop without jQuery-selection, except it will be impossible to move several items at the same time because the selection with the Ctrl/Shift keys will be not loaded. In the same way jQuery-tabs can function alone. However without jQuery-dragndrop it will be impossible to reorganize the tabs with the mouse.
